# User Personas

WorkSphere AI is designed for multiple types of users. Each user has different responsibilities and permissions within the platform.

---

# 1. Platform Owner

## Description

The Platform Owner manages the entire WorkSphere AI platform.

This user belongs to the SaaS provider (our company), not to any customer company.

## Responsibilities

- Manage companies
- Verify companies
- Manage subscriptions
- Enable/Disable features
- Monitor platform health
- View global analytics
- Manage platform administrators

## Permissions

- Full access
- Manage all tenants
- Access system monitoring
- View audit logs

---

# 2. Company Administrator

## Description

The Company Administrator is responsible for configuring the platform for a specific company.

Usually this role belongs to HR Directors or IT Administrators.

## Responsibilities

- Configure company
- Create departments
- Create teams
- Invite employees
- Assign managers
- Configure permissions
- Manage company settings

## Permissions

- Full company access
- Cannot access other companies
- Manage company users

---

# 3. HR Manager

## Description

Responsible for recruitment and employee management.

## Responsibilities

- Create job openings
- Review candidates
- Schedule interviews
- Send offers
- Manage onboarding
- Assign learning paths
- View HR reports

## Permissions

- Recruitment
- Employees
- Learning
- Analytics

---

# 4. Manager

## Description

Responsible for managing a team.

## Responsibilities

- View team members
- Track onboarding
- Review learning progress
- Give recognition
- Approve Employee of the Month
- View team analytics

## Permissions

- Team management
- Recognition
- Reports

---

# 5. Employee

## Description

A company employee using the platform.

## Responsibilities

- Complete onboarding
- Update profile
- Participate in events
- Complete learning paths
- Take quizzes
- Receive certificates
- Participate in discussions
- Give recognition

## Permissions

- Personal profile
- Learning
- Community
- Events
- Recognition

---

# 6. Candidate

## Description

A job applicant applying for open positions.

## Responsibilities

- Create profile
- Upload resume
- Apply for jobs
- Track applications
- Schedule interviews
- Accept or reject offers

## Permissions

- Personal profile
- Job applications
- Interview schedule
